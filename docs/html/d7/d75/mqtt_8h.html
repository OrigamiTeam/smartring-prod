◆ mqtt_message_type_t

enum mqtt_message_type_t

Type of MQTT message to create the correct message and topic.

Enumerator
SEND_SENSORS_RESPONSE 

Send the sensors value when requested.

Topic : d/{device_mac}/wl Info : Sensor raw value

SEND_DEVICE_INFORMATION 

Send the device information, when connected.

Topic : d/{device_mac}/con Info : no deposit full deposit wifi rssi Firmware version connection type

SEND_CALIBRATION 

Send device calibration.

Topic : d/{device_mac}/c Info : no deposit full deposit

SEND_SENSORS 

Send device sensors.

Topic : d/{device_mac}/wl Info : Sensors raw value

SEND_LOGIN 

Send device inserted login pin.

Topic : d/{device_mac}/pin Info : Inserted pin

SEND_ORDER_MODE 

Send device order mode.

Topic : d/{device_mac}/o/mode Info : Device order mode

SEND_STOCK 

Send device stock.

Topic : d/{device_mac}/stock Info : Stock

SEND_TICKET 

Send support ticket.

Topic : d/{device_mac}/t Info : Support type id number {smart_ring_support_type_t}

SEND_NEW_DELIVERY 

Send new delivery request.

Topic : d/{device_mac}/o Info : Number of bottles Number of cups

SEND_CONFIRM_DELIVERY 

Confirm new delivery.

Topic : d/{device_mac}/o/conf Info : Transaction ID Confirmation character "a"

SEND_GET_ORDERS 

Get orders.

Topic : d/{device_mac}/o/list Info : Empty (Since the library doesn't allow, send a space " ")

SEND_REQUEST_DEVICE_INFORMATION 

Request device information.

Topic : d/{device_mac}/s Info : 0 (Number 0 for device)

SEND_REQUEST_GROUP_INFORMATION 

Request group information.

Topic : d/{device_mac}/s Info : 1 (Number 1 for group)
