◆ ACCEPTED_INTERVAL

#define ACCEPTED_INTERVAL   240

need to differ from the last_sensor_sent by ACCEPTED_INTERVAL The value 240 is considered as about 1 Liter as requested by Marco Dias.

Considering:

Full tank (Raw value) : 89711
Empty tank (Raw value) : 85179
Difference = 89711 - 85179 = 4532
1 Bottle = 18.9 Liters = 4532
1 Liter = 4532 / 18.9 = 239.78 = 240 (Raw)

◆ STABILIZED_INTERVAL

#define STABILIZED_INTERVAL   15

The value is considered stable if the variation is below the STABILIZED_INTERVAL
